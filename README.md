# Price Monitor & Alert System (Scraper & Alert System)

Высоко масштабируемая микросервисная система мониторинга цен на товары маркетплейса Wildberries с автоматическим уведомлением пользователей в Telegram при падении цены ниже целевого уровня.

## Архитектурные особенности

*   **Микросервисная архитектура:** Разделение на два независимых сервиса (`Core Service` и `Scraper Service`), что позволяет масштабировать ресурсоемкий парсинг отдельно от основного API.
*   **Асинхронные очереди (RabbitMQ):** Задачи на парсинг не блокируют основной поток API. Брокер сообщений гарантирует доставку задач воркерам с поддержкой ручного подтверждения (`Manual Acknowledgment / noAck: false`).
*   **Паттерн Strategy (Стратегия):** Движок парсера спроектирован через абстрактные интерфейсы, позволяя легко добавлять новые маркетплейсы без изменения основного кода системы.
*   **Пакетная обработка (Batching):** Cron-планировщик обрабатывает базу данных порциями (`take/skip`), исключая перегрузку оперативной памяти (OOM) при миллионах записей.
*   **Транзакционность:** Обновление цен и запись в историю (`PriceHistory`) захеджированы через атомарные транзакции базы данных PostgreSQL.
*   **Антифрод-маскировка:** Singleton-процесс Puppeteer, который инициализирует сессию на реальной странице товара для автоматического прохождения JS-челленджей защиты маркетплейса. Вместо нестабильного разбора DOM-верстки, данные мгновенно собираются через AJAX-инжект (window.fetch) прямо из доверенной среды браузера. При фиксации блокировок доступа (ошибки 498) система не падает, а через протокол CDP автоматически затирает кэш сессии, имитирует активность человека и выполняет успешный повторный запрос, гарантируя отказоустойчивую обработку очереди RabbitMQ.

## Технологический стек

*   **Backend:** NestJS (v10+), TypeScript
*   **Database & ORM:** PostgreSQL, Prisma ORM
*   **Broker:** RabbitMQ
*   **Scraping Engine:** Puppeteer (Headless Chromium) + Chrome DevTools Protocol
*   **Alerts:** Telegraf (Telegram Bot API)
*   **Infrastructure:** Docker, Docker Compose

## Инструкция по запуску
### 1. Запуск инфраструктуры через Docker Compose
Находясь в корневой папке проекта, запустите базу данных и брокер сообщений, явно указав путь к файлу конфигурации из `core-service`:

```bash
docker compose --env-file ./core-service/.env up -d
```

### 2. Подготовка базы данных (Prisma)
Перейдите в папку основного сервиса, установите зависимости и примените миграции для развертывания структуры таблиц и индексов в PostgreSQL:

```bash
cd core-service
npm install
npx prisma migrate dev --name init
```

Запустите Core-сервис в режиме разработки:
```bash
npm run start:dev
```

### 3. Запуск микросервиса парсинга
Откройте второй терминал, перейдите в папку скрапера, установите зависимости и запустите воркер очереди:

```bash
cd ../scraper-service
npm install
npm run start:dev
```

## Сценарий сквозного тестирования

Для проверки работоспособности асинхронной цепочки без ожидания реального изменения цен на маркетплейсе выполните следующие действия:

1.  Найдите созданного бота в Telegram, запустите его командой `/start` и узнайте свой `telegramId` (через `@userinfobot`).
2.  Отправьте `POST`-запрос на добавление любого товара с Wildberries:
    *   **URL:** `POST http://localhost:3000/products`
    *   **Body (JSON):**
        ```json
        {
          "url": "https://www.wildberries.ru/catalog/123456789/detail.aspx?targetUrl=MI"
        }
        ```
    *   *Ответ вернет `id` созданного товара в нашей системе.*
3.  Создайте подписку на уведомление (Алерт), указав заведомо завышенную целевую цену, чтобы триггер сработал мгновенно:
    *   **URL:** `POST http://localhost:3000/alerts`
    *   **Body (JSON):**
        ```json
        {
          "productId": "id_товара_из_предыдущего_ответа",
          "targetPrice": 50000,
          "telegramId": "ваш_telegram_id"
        }
        ```
4.  **Результат:** Планировщик отправит задачу в RabbitMQ, `Scraper Service` инициализирует сессию, безопасно извлечет цену, вернет её в `Core Service` по вебхуку, база данных запишет транзакцию, а бот пришлет вам сообщение в Telegram со ссылкой на товар.
